    Web Crawler

    Web crawler adalah program yang berjalan menggunakan metode tertentu dan secara otomatis mengambil data atau mengumpulkan beberapa data informasi yang ada di dalam suatu website. Web crawler akan mengunjungi alamat website yang dicantumkan, kemudian mengambil dan menyimpan data yang terkandung dalam website tersebut. Proses web crawler dalam mengunjungi setiap dokumen web disebut dengan web crawling.

    Install library:
    • install requests : pip install requests

    Requets adalah modul pada pythin yang dapat digunakan untuk mengirim request HTTP.

    • install beautiful soup 4 : pip install bs4

    Beautiful Soup 4 adalah library pada python yang digunkan untuk mengambil data dari file HTML.
